# Collaborative Code Editor

A real-time collaborative code editor that allows multiple users to write and edit code simultaneously. Built using React.js, Node.js, Express.js, and Socket.IO.

## Features

- Real-time code synchronization
- Multi-user collaboration
- Responsive user interface
- Fast communication using WebSockets
- Client-server architecture
- Easy to extend for multiple programming languages

## Tech Stack

- React.js
- Node.js
- Express.js
- Socket.IO
- JavaScript
- HTML
- CSS

## Project Structure

```
client/
server/
src/
```

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
cd client
npm install
npm run dev
```

### Run Backend

```bash
cd server
npm install
npm start
```
